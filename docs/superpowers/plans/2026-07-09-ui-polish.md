> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

- New i18n keys → both `en.ts` and `pt.ts`.

- [ ] **Step 1: Add `BackIcon`**

In `src/ui/icons.tsx`:
```tsx

  <Icon className={className}><path d="M15 5l-7 7 7 7" /></Icon>

```

- [ ] **Step 2: Add `onBack` to `ScreenHeader`**

Replace `src/ui/ScreenHeader.tsx` with:
```tsx
import type { ReactNode } from 'react'
import SectionLabel from './SectionLabel'
import { BackIcon } from './icons'

export default function ScreenHeader({
  label, title, actions, onBack,
}: { label: string; title: string; actions?: ReactNode; onBack?: () => void }) {
  return (
    <div className="mb-5 flex flex-wrap items-end justify-between gap-3">
      <div className="flex items-center gap-2">
        {onBack && (
          <button
            type="button"
            onClick={onBack}
            aria-label="Back"
            className="rounded-md p-1.5 text-ink-muted hover:text-ink focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-accent"
          >
            <BackIcon />
          </button>
        )}
        <div>
          <SectionLabel>{label}</SectionLabel>
          <h1 className="mt-0.5 text-xl font-semibold tracking-tight">{title}</h1>
        </div>
      </div>
      {actions && <div className="flex flex-wrap items-center gap-2">{actions}</div>}
    </div>
  )
}
```
(The `aria-label="Back"` is a plain string so `ScreenHeader` stays i18n-free like today; ClubScreen already carries the translated `club.back` string, but the icon's label can stay literal, or pass a translated one — keep it simple with the literal for the shared component.)

- [ ] **Step 3: Wire ClubScreen**

In `src/screens/ClubScreen.tsx`, pass `onBack` to `ScreenHeader` and drop the right-side text button:
```tsx
      <ScreenHeader
        label={t('club.position', { position: positionOf(state, teamId), division: team.division })}
        title={team.name}
        onBack={onBack}
      />
```
Remove the now-unused `actions={<Button …>{t('club.back')}</Button>}` and, if `Button` is no longer used elsewhere in the file, drop its import. Keep the `club.back` key (still referenced? if not, it may be retired — check with a search).

- [ ] **Step 4: Typecheck + manual**

Run: `npx tsc -b --force`
Expected: no errors.
Manual: opening a club shows a back arrow at the top-left; clicking it returns to the previous screen.

Compute `lowestDivision` once (generic over division count):
