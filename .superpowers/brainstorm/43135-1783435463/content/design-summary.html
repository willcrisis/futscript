The redesign at a glance — "Quiet Heritage"

Everything we decided, on one screen. Approve or adjust in the terminal.

Foundations

Tailwind v4 · warm paper + stone ink, hairline rules · green = the only accent (your club, money, positive) · Inter for prose, JetBrains Mono for every number (self-hosted, offline-safe) · uppercase micro-labels · dark mode = warm stone night, sidebar toggle, persisted.

Shell

Desktop: slim sidebar (nav + club vitals + theme toggle + Advance Week always reachable). Mobile: bottom tab bar (Home, Squad, Table, Finance, More-sheet) with floating Advance. Consistent screen headers: uppercase label, big title, actions right.

Home — new command center

Next fixture card · 5-row table excerpt centered on you (green spine) · cash + weekly delta + hand-rolled sparkline · fan mood · attention list (offers, expiring contracts, construction, board patience) · big Advance Week.

Component kit (~10 primitives)

Panel · SectionLabel · DataTable (desktop table ↔ automatic mobile card list from the same column defs) · StatChip · Button · Badge (injury/suspension/form) · ConfirmButton (two-click) · Toast (offer arrived, expansion done…) · EmptyState · MoneyText. No component library unless a real a11y gap appears.

Screens

All ten migrate; engine untouched. Squad = flagship table with sticky group headers + badges. Match ticker gets the glow-up: huge mono scoreline, minute progress bar, icon event feed, 1×/2×/skip. Cup = bracket-flavored rounds. Finance = stat cards + week-grouped ledger. Saves = slot cards. Champion/sacked = full-bleed moments.

Quality & non-goals

AA contrast both themes, focus rings, reduced-motion respected, keyboard-friendly tables. 146 engine tests stand as the safety net; per-screen browser verification + dual-form-factor acceptance. No router lib, chart lib, engine edits, or new game features.

Delivery phases

A: tokens + fonts + kit + theme toggle → B: shell + Home → C: screens in pairs → D: ticker + banners → E: delete old CSS, a11y sweep, acceptance.
